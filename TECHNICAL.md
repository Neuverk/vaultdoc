> AI-powered compliance documentation platform by Neuverk

## Project Overview

Vaultdoc is a multi-tenant SaaS platform that allows companies to create, manage,
approve and publish compliance documentation (SOPs, Policies, Runbooks etc.)
using AI. It maps documents to compliance frameworks like ISO 27001:2022, TISAX,
GDPR, ITIL v4, SOC 2 and more.

## Authentication & Authorization

### Provider: Clerk
- Handles all authentication (email, Google, Microsoft Entra/Azure AD)
- JWT tokens verified server-side on every request
- MFA planned for Phase 3

### Middleware (`middleware.ts`)
Protects all routes except `/`, `/sign-in`, `/sign-up`.
Any unauthenticated request to a protected route is redirected to `/sign-in`.

### Role hierarchy

superadmin  → Neuverk staff only. Sees all tenants.
admin       → Company admin. Manages their tenant.
manager     → Department manager. Approves documents.
author      → Creates documents. Sees own docs only until released.
viewer      → Read-only access to released documents.

### Document visibility rules
- Authors can only see their own documents
- Once a manager releases a document as "internal", all department members see it
- "Public" documents are visible to all users in the tenant
- "Confidential" requires explicit access grant
- "Restricted" requires admin approval to view
